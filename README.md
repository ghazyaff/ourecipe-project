## Description
This project is built using :
* Laravel 8 with Jetstream and Livewire
* TailwindCSS
* AlpineJS

## Requirements
* PHP 7+
* NodeJS

## Installation
* Edit .env file according to your needs
* Run :
```bash
$ composer update
$ php artisan key:generate
$ php artisan storage:link
$ php artisan migrate:fresh
$ php artisan serve
```
